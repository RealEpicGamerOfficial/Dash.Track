# Dash.Track
Dash.Track is a local ToF (Time of Flight) simulator that uses a locally ran "intelligence model" to detect objects, people, and more! FOR TESTING ONLY, PLEASE NOTE DATA MAY BE SHARED WITH 3rd PARTIES! 

## About this project
* Detect objects LIVE with a local "intellegence model"
* Know what is possable with the most cheap cameras or expensive cameras
* Protect against attacks and improve protection
* Test "Noise-Protection" and "Light-Based-Protections" to Version 1 of Dash.Track [Beta]

## Tools
* ToF Sensor Simulation
* Skeletion Detection (BETA)
